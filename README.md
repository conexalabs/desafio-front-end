# Localizador de Empresas

Demo: https://localizador-empresas.herokuapp.com

## Introdução
Aplicação feita em [Vue.js](http://vuejs.org) para localizar empresas através do CNPJ.

### Development Setup

``` bash
# install deps
npm install

# build dist files
npm run build

# serve examples at localhost:8000
npm run dev

# lint & run all tests
npm run test
```
